# Must Practice Questions

This folder contains recurring LeetCode questions aggregated from company-wise lists in this repository.

## Methodology

- Source scanned: each company folder's 5. All.csv.
- Deduplication key: LeetCode Link when present, otherwise normalized Title.
- Count means the number of distinct company folders where the question appears.
- Rows are sorted by Count, then AverageFrequency, then title.

## Files

- `common-questions.csv`: all questions repeated across at least 2 companies.
- `must-practice-top-150.csv`: the highest-repeat subset to practice first.
- `topic-summary.csv`: topic-level summary over repeated questions.
